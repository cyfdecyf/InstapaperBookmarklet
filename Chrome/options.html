InstapaperBookmarklet

Feature

Add a button in the toolbar,

  • Single click: run Instapaper bookmarklet on current page
    • If current page has no URL, open Instapaper unread page
  • Double click: open Instapaper unread page in new tab

Please set your personal Instapaper bookmarklet

  1. Open Instapaper extras page (make sure you have logged in)
  2. Right click on the bookmarklet, copy link address
  3. Paste below and then save
